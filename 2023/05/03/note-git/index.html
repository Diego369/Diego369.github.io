修改最后一次提交 commit 的信息

修改历史提交 commit 的信息

操作步骤：

  • git rebase -i 列出 commit 列表
  • 找到需要修改的 commit 记录，把 pick 修改为 edit 或 e，:wq 保存退出
  • 修改 commit 的具体信息git commit --amend，保存并继续下一条git rebase --continue，直到全部完成
  • 中间也可跳过或退出git rebase (--skip | --abort)

修改完成后，强制推送到远程服务器
